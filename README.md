# txp-swagger

mock 数据配置界面化、通过 swagger 生成 mock 数据、通过 swagger 生成自定义代码

## 开发计划

### mock 数据配置界面化

#### 前端

1. 主界面：显示项目，新增项目按钮、弹窗。
2. 项目详情页面：显示项目内接口列表、

#### 后端

1. 主界面：
   > 项目显示接口：[{name}]
   > 新增项目接口：{name}
2. 项目详情页面：
   > 项目详情列表显示接口：[{Method,URL,description,name}]

## 目录说明

node：后端服务。
react：前端服务。
